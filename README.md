# 不動産営業向け 風水説明レポートAI

間取り図をClaude Visionで読み取り、不動産営業担当者が顧客へ説明するための穏やかな風水参考レポートを生成するNext.jsアプリです。

## セットアップ

```bash
npm install
cp .env.example .env.local
```

`.env.local` の `ANTHROPIC_API_KEY` にClaude APIキーを設定し、次を実行します。

```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

APIキーを設定していない場合でも、「サンプルレポートを見る」から、間取り図と完成レポートの一連の表示を確認できます。このサンプル機能はClaude APIを呼び出さず、料金は発生しません。

## 料金ゼロの公開デモ

本番ビルドでは、初期状態で実画像のAI解析を停止します。画面はサンプル専用表示になり、解析APIへ直接リクエストしてもClaude APIは呼ばれません。Vercelへサンプルデモとして公開する場合は、次の環境変数を未設定または `false` のままにし、`ANTHROPIC_API_KEY` も登録しないでください。

```text
ENABLE_LIVE_ANALYSIS=false
NEXT_PUBLIC_ENABLE_LIVE_ANALYSIS=false
```

将来、認証・共有レート制限・利用上限を整備して実解析を公開する場合だけ、サーバー用と画面用の両方を `true` にします。ローカルの `npm run dev` では、この公開デモ設定にかかわらず実解析を利用できます。

北マークが小さい場合や図面に記載がない場合は、入力画面の「北方向の指定」で、斜めを含む8方向から北を手動補正できます。色付きの注釈矢印は方位磁針と区別するようClaudeへの指示にも含めています。

AIによる北方向の自動認識は参考判定です。自動認識で作成したレポートには確認警告を表示し、手動補正して再生成できるようにしています。営業資料として使用する前に、図面の方位記号と必ず照合してください。

## セキュリティとプライバシー

- APIキーはサーバー環境変数だけで扱い、ブラウザへ公開しません。
- JPEG、PNG、WebPのみ、最大4MBに制限し、MIMEタイプとファイル署名を検査します。
- Vercel Functionsの4.5MB制限を超えないよう、リクエスト全体のサイズも事前検査します。
- 解析APIは同一生成元を確認し、簡易レート制限を行います。
- AI出力はZodで構造と最大長を検証してから画面へ渡します。
- Claude APIの自動再試行を無効化し、テスト中の意図しない重複課金を防ぎます。
- 本番では2つの明示的な機能フラグが揃わない限り、実解析を停止します。
- 間取り画像や生成レポートをアプリ内のデータベースへ保存しません。
- セキュリティヘッダー、CSP、APIレスポンスの `no-store` を設定しています。
- 画像内の文字は信頼できないデータとして扱い、プロンプトインジェクション対策をプロンプトへ含めています。

注意：現在のレート制限は単一プロセス内の簡易実装です。本番で複数インスタンスを使用する場合は、Vercel FirewallまたはRedis等の共有ストアへ置き換えてください。認証機能も本番公開前に追加してください。
